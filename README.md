# Kubernetes EC2 Operator (Portfolio MVP)

A Kubernetes Operator built with **Kubebuilder** and **Go** that declaratively manages AWS EC2 instances via a Custom Resource `EC2Instance`.

- Follows freeCodeCamp full course: [Build Your Own Kubernetes Operators with Go and Kubebuilder](https://www.youtube.com/watch?v=odP153inZUo)
- Reference implementation: [shkatara/kubernetes-ec2-operator](https://github.com/shkatara/kubernetes-ec2-operator)
- Goal: Production-ready MVP with idempotency, finalizers, AWS SDK integration, Helm packaging, and clean RBAC.
